Janus - Software Engineering Diagramming Tool

2017-04-30

I've been putting some of my free time into a new open source project called Janus. It is meant to be a diagramming tool for various activities around software projects. Frankly, I am just writing this tool for myself.

Imagine the following very common scenario. You go into a meeting with some new clients. Things run very smoothly. They are excited about getting their product from you. You are excited about building their product for them. You decide that you are going to start working on this as soon as the paperwork is done. And after doing some basic requirements analysis at the start, you discover that you didn't even ask that very basic question that you need to start.

Well, it is not really your fault. Normal human brain does not work in detailed lists. The problem is more the fact that the you did not do your analysis early enough.

Here is a tool for doing so! I am hoping to turn it into something that can greatly assist engineers in real time in generating questions and doing basic analysis to make these sessions more effective.

It still has a long way to go.
